Добре дошли в курса "Търсене и извличане на информация"

Ще използваме Python през целия курс. Ако вече имате добра инсталация на Python, чудесно! Но се уверете, че това е поне Python версия 3.5. Ако не, най-лесното може да си създадете подходяща среда по следния начин:

# 1. Уверете се, че имате 2-3 GB свободно дисково пространство на вашия компютър и след това се насочeте към (https://www.anaconda.com/download/) за да инсталирате версията на Anaconda на Python 3. Има готови инсталационни пакети за масово използваните операционни системи (Windows, Linux, MacOS).

# 2. След като инсталирате conda, затворете всички отворени терминали, които имате. След това отворете нов терминал.


# 3. Свалете файла tii.yml от Moodle и го запишете в текущата директория на вашия терминал.

# 4. Създайте среда със зависимости, посочени в tii.yml като изпълните в терминала следната команда:
    
    conda env create -f tii.yml

# 5. Активирайте новата среда:
    
    conda activate tii



След завършване на работа може да деактивирате средата с командата:

    conda deactivate
